# Content

The **content library** holds all media files used on the platform: images, videos, HTML5 packages, PDFs, web URLs, presentations, menu boards, zone layouts.

## Where to find content
Workspace sidebar → **Content**. URL: `/workspaces/:workspaceId/content`.

## Uploading content
1. Click **Upload** (top right).
2. Drop files or browse — multiple files at once is fine.
3. Supported types:
   - **Images**: JPG, PNG, WebP, GIF, SVG
   - **Videos**: MP4, WebM, MOV (auto-transcoded to optimised MP4)
   - **HTML5**: ZIP packages with an `index.html` entry point
   - **PDF**: rendered as a slideshow at a configurable page duration
   - **Web URL**: any HTTPS URL displayed in a kiosk browser
4. Add **tags** during upload to make content easier to find later.

## Organising content
- **Folders** — Workspace sidebar → **Content** → create folders and drag items in.
- **Tags** — multiple tags per item; great for cross-folder filtering.
- **Smart Views** — saved filter combinations.

## Versions
Replacing a file creates a new **content version**. Old versions remain accessible — useful for rollback.

## Thumbnails & previews
Thumbnails are generated automatically. Click any content item to preview it in the right panel.

## Approval & expiry
Content can optionally require approval before being added to playlists. Set an **expiry date** to auto-remove time-sensitive content (e.g. promotions).
